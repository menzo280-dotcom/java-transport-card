# java-transport-card
# 🚍 Sistema de Bilhete de Ônibus / Bus Card System

Projeto simples em Java que simula o pagamento de passagens de transporte público utilizando Programação Orientada a Objetos.

Simple Java project that simulates public transport fare payment using Object-Oriented Programming.

---

## 📌 Funcionalidades / Features

- Usuário comum paga tarifa inteira  
  Regular user pays full fare

- Estudante paga meia tarifa  
  Student pays half fare

- Idoso possui isenção  
  Elderly users are exempt from payment

- Simulação de saldo e pagamento  
  Balance and payment simulation

---

## 🧠 Conceitos utilizados / Concepts used

- Programação Orientada a Objetos (POO) / OOP  
- Herança / Inheritance  
- Polimorfismo / Polymorphism  
- Classes abstratas / Abstract classes  
- @Override

---

## 🏗️ Estrutura do projeto / Project structure

- Usuario (classe abstrata / abstract class)  
- Estudante / Student  
- Idoso / Elderly  
- Comum / Regular user  
- Program (classe principal / main class)
